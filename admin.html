🔧 Discord Webhook URL

Masukkan URL webhook Discord kamu agar transaksi otomatis terkirim ke channel Discord.
Cara buat: Discord → Edit Channel → Integrations → Webhooks → New Webhook → Copy Webhook URL

📖 Panduan

  1. Pergi ke server Discord kamu
  2. Klik kanan channel yang dituju → Edit Channel
  3. Klik Integrations → Webhooks
  4. Klik New Webhook → beri nama → Copy Webhook URL
  5. Paste URL di kotak di atas → Simpan
  6. Sekarang setiap transaksi baru akan otomatis terkirim ke Discord!
